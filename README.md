# Basic Port Scanner

 Description
A simple Python-based port scanner that identifies open ports on a target system using socket programming.

 Objective
To understand how network ports work and how open ports can be detected for basic security analysis.

Technologies Used
- Python (socket module)

Features
- Scans ports in a defined range (1–100)  
- Detects open ports on a target IP address  
- Uses TCP connection attempts for scanning  
- Displays results in a simple format  

 How to Run
1. Make sure Python is installed  
2. Download or clone this repository  
3. Open terminal/command prompt  
4. Run the script:
   ```bash
   python port_scanner.py.
